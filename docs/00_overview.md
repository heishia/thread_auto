# Overview

## Project Summary

ThreadAuto is a desktop application designed to automate the creation of Threads SNS posts. It uses Google Gemini AI to generate engaging content following a specific persona and writing style.

## Problem Statement

Creating consistent, engaging social media content is time-consuming. ThreadAuto automates this process while maintaining a specific brand voice and style.

## Solution

A desktop application that:
- Generates posts using AI with customizable prompts
- Supports multiple post types for different purposes
- Allows manual or automatic generation
- Provides an easy copy-to-clipboard workflow

## Target Users

Content creators and social media managers who want to maintain an active Threads presence with minimal effort.
